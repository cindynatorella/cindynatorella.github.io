Creation and design of search engine based on images from Pompei. Integration of search features such as textual similarity, image similarity as well as a combination of both. It was made using image processing, computer vision, and natural language processing technology.

Lead Software Engineer on A House of the Ozarks, a 3D virtual tour Unity application featuring the work of architect Fay Jones. Integrated touch-base and click-based mechanics, UI system, and event system. Coded for guided first-person view and third-person view.

Software Engineer on Forum Boarium, a 3D archaeological visualization Unity application for presenting information on the ancient Forum Boarium site in Rome. Primary responsibilities focused on UI/UX and gameplay mechanics development.

Technology used: C#, Unity, Inky, Docker, Git, C++, Python, SQL, OpenCV, Keras, Colab Notebook, TensorFlow.
